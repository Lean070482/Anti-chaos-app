🛡️ Anti Chaos - Finanças sob Controle
O Anti Chaos é um aplicativo de gestão financeira pessoal desenvolvido em Flutter, focado em simplicidade, velocidade e organização. Ele foi projetado para quem quer dominar o fluxo de caixa, eliminar gastos desnecessários e planejar objetivos de vida sem a complexidade de planilhas pesadas.

🚀 Funcionalidades
Fluxo de Caixa Reativo: Visualize seu saldo real (Salário + Renda Extra - Gastos) em tempo real.

Gestão de Gastos Inteligente:

Lançamento rápido por categoria.

Detalhamento personalizado para a categoria "Outros".

Marcação de gastos fixos vs. variáveis.

Raio-X Financeiro: Gráficos de barras e indicadores de impacto que mostram onde seu dinheiro está "sangrando".

Alerta visual (cor vermelha) para categorias que consomem mais de 40% da renda.

Jornada de Conquistas: Sistema de metas para poupar dinheiro com barra de progresso e aportes manuais.

Gestão de Fluxo: Remova registros incorretos com um simples gesto de deslizar para o lado.

Persistência Local: Seus dados são salvos no dispositivo usando o banco de dados de alta performance Hive.

🛠️ Tecnologias Utilizadas
Flutter & Dart: UI moderna e fluida.

Hive: Armazenamento local rápido e eficiente.

FL Chart: Visualização de dados e gráficos analíticos.

Intl: Formatação de datas e moedas.
