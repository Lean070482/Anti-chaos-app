
# 🛡️ Anti-Chaos | Gestor Financeiro Premium

O **Anti-Chaos** é um aplicativo de gestão financeira pessoal desenvolvido em Flutter, projetado para quem busca controle total sobre suas finanças sem a confusão visual e os anúncios dos aplicativos tradicionais. O projeto foca no conceito de "Finanças Conscientes", separando o que é essencial do que é desejo.

## ✨ Funcionalidades Principais

- **Dashboard Clean:** Interface Black & Gold focada no saldo real e leitura rápida.
- **Gestão de Prioridades:** Separação inteligente entre gastos essenciais (sobrevivência) e gastos variáveis.
- **Módulo de Metas (Dreams):** Planeje a compra de uma moto ou uma viagem com barras de progresso que mostram o quanto falta para realizar seu objetivo.
- **Gráficos de Categoria:** Visualização intuitiva da distribuição dos seus gastos mensais.
- **Privacidade Total:** Todos os dados são armazenados localmente no dispositivo (Offline-first).

## 🛠️ Tecnologias Utilizadas

- **[Flutter](https://flutter.dev/):** Framework para desenvolvimento multiplataforma.
- **[Hive](https://pub.dev/packages/hive):** Banco de dados NoSQL ultrarrápido para persistência local.
- **[FL Chart](https://pub.dev/packages/fl_chart):** Biblioteca para criação de gráficos dinâmicos.
- **[Intl](https://pub.dev/packages/intl):** Internacionalização e formatação de moedas/datas (BRL).
